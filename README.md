# Attendance Tracker

A modern, responsive attendance tracking web application built with HTML, CSS, and JavaScript.

## Live Demo

🌐 [View Live on Netlify](https://attendance-tracker-app.netlify.app)

## Login Credentials

|----------|--------------|---------------|

> You can also create your own account using the **Sign Up** page.

## Features

- User authentication (signup/login)
- Dashboard with attendance statistics
- Daily attendance tracking with status (Present, Absent, Late)
- Weekly attendance visualization with charts
- Monthly attendance summary
- Export functionality for attendance reports
- Responsive design that works on all devices

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS classes)
- JavaScript (ES6+)
- Chart.js for data visualization
- LocalStorage for client-side data persistence

## Setup Instructions

1. Clone the repository
   ```bash
   git clone https://github.com/Tejaswi1258/attendance-tracker.git
   ```
2. Open `index.html` in your browser
3. Sign up for a new account or log in with the credentials above
4. Start tracking your attendance!
